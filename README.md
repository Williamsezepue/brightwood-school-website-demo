# Brightwood School Website Demo

A polished one‑page school website demo built with **Next.js** and **Tailwind CSS**.  
Brightwood showcases modern layouts, responsive design, and interactive sections tailored for educational institutions.  
It emphasizes **accessibility, performance, and clarity** in presenting school programs, admissions, and faculty.

---

## Features

- **Responsive Layout** — Optimized for mobile, tablet, and desktop
- **Program Highlights** — STEM Lab, Creative Arts Studio, Sports & Wellness
- **Faculty & Admissions Sections** — Clear navigation and structured content
- **Performance** — Fast load times with Next.js SSR
- **Accessibility** — Semantic HTML and ARIA support
- **Modern Animations** — Smooth transitions powered by Framer Motion

---

## Tech Stack

- [Next.js](https://nextjs.org/) — React framework for production
- [Tailwind CSS](https://tailwindcss.com/) — Utility‑first CSS framework
- [TypeScript](https://www.typescriptlang.org/) — Static typing for reliability
- [Headless UI](https://headlessui.dev/) — Accessible UI components
- [Framer Motion](https://www.framer.com/motion/) — Animation library

---
